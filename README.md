## Quick start

### 1. Clone repository

```bash
git clone https://github.com/Brlaney/mtrx.git
```

### 2. Install dependencies

```bash
cd mtrx

# then for yarn users:
yarn install

# or npm users:
npm i 
```

### 3. Start development server

```bash
yarn dev

#or

npm run dev

```

Now navigate to [http://localhost:3000/](http://localhost:3000/) and check it out.
